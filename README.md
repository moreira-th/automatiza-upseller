# UpSeller Moreira

Extensão Chrome para automação no **UpSeller**. Arquitetura detalhada, padrões de código e guia para desenvolvimento em [AGENTS.md](./AGENTS.md).

## Instalação

1. `chrome://extensions` → **Modo do desenvolvedor**
2. **Carregar sem compactação** → selecionar `upseller-unificado/`
